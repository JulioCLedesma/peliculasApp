La aplicación de películas es un proyecto académico diseñado para gestionar un catálogo de películas de manera dinámica y funcional. Este sistema está desarrollado utilizando Angular 18 para el frontend y una API construida en Laravel para el backend. Su objetivo principal es permitir a los usuarios realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre una base de datos de películas, simulando una solución real utilizada en aplicaciones convencionales.
